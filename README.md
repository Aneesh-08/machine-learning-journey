# Machine Learning Journey

A daily log of my machine learning learning path — from fundamentals to hands-on implementation. Each folder represents one day of focused study, covering core concepts, code notebooks, and visualizations built from scratch and using libraries.

---

## 🎯 Objective

To build a strong foundation in Machine Learning by implementing algorithms step-by-step and understanding the mathematics behind them — one day at a time.

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Environment:** Jupyter Notebook
- **Libraries:** NumPy, Pandas, Matplotlib, Scikit-learn

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Aneesh-08/machine-learning-journey.git
   cd machine-learning-journey
   ```

2. **Install dependencies**
   ```bash
   pip install numpy pandas matplotlib scikit-learn jupyter
   ```

3. **Launch Jupyter Notebook**
   ```bash
   jupyter notebook
   ```

4. Open any day's folder and run the `.ipynb` notebook.
